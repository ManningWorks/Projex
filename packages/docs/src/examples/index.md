# Examples

Welcome to Folio examples! Here you'll find interactive demos and complete implementations to help you get started with Folio.

## Available Examples

- **[Full Portfolio](/examples/full-portfolio.md)** - A complete portfolio page with featured project, grid layout, and interactive filtering
- **[Filtered Grid](/examples/filtered-grid.md)** - A filterable project grid with type and status filters
- **[Searchable List](/examples/searchable-list.md)** - A searchable project list with real-time filtering
- **[Interactive Demo](/examples/interactive-demo.md)** - All interactive features combined in one playground

## Getting Started

Each example includes:

- Live React component previews
- Interactive controls to explore component options
- Real data fetched from GitHub and npm
- Code snippets you can copy

All examples use same components from Folio library, so you can easily adapt them to your needs.
