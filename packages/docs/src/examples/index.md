# Examples

Welcome to Folio examples! Here you'll find code samples and complete implementations to help you get started with Folio.

## Available Examples

- **[Solo Developer Portfolio](/examples/solo-developer.md)** - Mix of GitHub repos, npm packages, and manual projects in one portfolio
- **[Full Portfolio](/examples/full-portfolio.md)** - A complete portfolio page with featured project, grid layout, and interactive filtering
- **[Filtered Grid](/examples/filtered-grid.md)** - A filterable project grid with type and status filters
- **[Searchable List](/examples/searchable-list.md)** - A searchable project list with real-time filtering

## About These Examples

Each example includes:

- **Complete code snippets** you can copy into your project
- **HTML output** showing the structure Folio components generate
- **Styling examples** with CSS for data attributes
- **Configuration samples** for `folio.config.ts`

These examples demonstrate how to use Folio components and patterns. They are code samples and documentation, not live interactive previews.

All examples use the same components from the Folio library, so you can easily adapt them to your needs.
